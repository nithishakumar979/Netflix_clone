# 🧗 Chapter 4 — The Struggle

The biggest challenge was making the horizontal movie rows scroll smoothly while keeping every poster aligned correctly across different screen sizes.

Initially, the movie cards wrapped into multiple lines on smaller devices, breaking the design.

I solved this by:

- Using Flexbox with `overflow-x: auto`
- Preventing items from shrinking using `flex-shrink: 0`
- Hiding scrollbars while keeping scrolling functional
- Adding responsive media queries
- Fine-tuning card widths for tablets and mobile devices

Another challenge was implementing the sticky navigation bar that changes color on scroll without affecting performance. JavaScript's `scroll` event combined with CSS transitions solved this efficiently.
